# CharacterBuilder
Pathfinder Character builder using c++

These instructions are going to be for Linux
Machines only.

Before compiling you want to have fltk install
on your machine.  You can install this
software with the following command in your
terminal:
sudo apt-get install libfltk1.3-dev


Before running you need to compile the program
using the following line in your terminal
(make sure you are in the correct directory):
g++ -std=c++14 -Wall MAIN.cpp -o PROJECT -lfltk

You can then run the file using the command:
./PROJECT
